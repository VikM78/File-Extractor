# Universal File Extractor

[![Python 3.6+](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Универсальный инструмент для извлечения файлов из текстовых шаблонов с поддержкой комментариев и интерактивным режимом.

## 📋 Описание

`extract.py` - это легковесный инструмент для распаковки файлов из текстовых шаблонов. Он распознает специальные маркеры `start_my_file`/`end_my_file` с комментариями, поддерживает множество языков программирования и форматов файлов.

### Основные возможности

- 🔍 **Автоматическое распознавание** маркеров с комментариями
- 📁 **Поддержка 30+ форматов** файлов
- 🎯 **Интерактивный режим** для удобной работы
- 🚀 **Неинтерактивный режим** для автоматизации
- 🔄 **Поддержка множества источников**
- 📝 **Гибкая обработка** шаблонов
